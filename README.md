# plassikortittaja
A simple script for automatic table card creation from .csv-files. Needs pandas to funciton. Takes .csv and the file to be created as arguments, for example ```python plassikortittaja.py vappu/ilmo_tekis.csv vappu/tekis_kortit``` would read ```ilmo_tekis.csv``` from the folder ```vappu``` and create ```tekis_kortit.html``` in the same folder.
